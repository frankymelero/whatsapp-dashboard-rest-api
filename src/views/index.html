Welcome to Sudoku API

Endpoints

Endpoint Method Description Params
/ GET Main API page NO
/api/v1/sudoku/get-all GET Retrieves all Sudokus in the DB NO
/api/v1/sudoku/get-by-difficulty/:difficulty GET Retrieves all Sudokus with the received difficulty YES
/api/v1/sudoku/get-one/:id GET Retrieves a sudoku with the received id YES
/api/v1/sudoku/get-random/ GET Retrieves a random Sudoku with random difficulty NO
/api/v1/sudoku/get-random-by-difficulty/:difficulty GET Retrieves a random sudoku with received difficulty YES

Additional information

· 5 new sudokus are added daily via Cron Job.

· /get-all endpoint will be deactivated upon 300 registers. Once We reach the limit, the data received will be paginated and the endpoint will be modificated to select the page and number of results.
